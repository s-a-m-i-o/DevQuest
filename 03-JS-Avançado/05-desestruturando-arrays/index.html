Desestruturação JavaScript

A DESESTRUTURAÇÃO é uma expressão Js que nos permite extrair variaveis de objetos ou arrays

CHATGPT - A desestruturação (ou destructuring) no JavaScript é uma forma prática de extrair dados de arrays ou objetos e atribuí-los a variáveis de maneira mais concisa. Isso permite que você acesse elementos ou propriedades de forma mais direta, sem precisar acessar manualmente cada item ou propriedade.
