O que é API

Application Programming Interface

Ou, Intrerface de Programação de Aplicações


É um conjunto de padões de programação que permite a construção de sistemas que abstraem muita coisa de código

Geralmente essas funcionalidades vão ser disponbilizadas atraves de endpoints

Endpoints

Endpoints nada mais são que urls, edenreços de web que a gente vai chamar para por exemplo buscar uma lista de clientes, ou para salvar um novo cliente em um banco de dados.

  • GET - quando queremos buscar uma informação
  • POST - serve para enviar uma informação
  • PUT - serve para alterar o estado de alguma informação
  • DELE - serve para deletar informações
