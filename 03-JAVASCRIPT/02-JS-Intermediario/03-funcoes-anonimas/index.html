Funções anonimas

Uma função anônima é uma função sem nome.
Elas são úteis quando você quer criar uma função rapidamente, muitas vezes para ser usada apenas uma vez, ou para passar como argumento para outra função.

Como funciona uma funçã anonima?

Em vez de dar um nome para a função, você define a lógica diretamente.
No JavaScript, você pode criar uma função anônima de várias formas, mas a mais comum é atribuí-la a uma variável ou usá-la como argumento.
