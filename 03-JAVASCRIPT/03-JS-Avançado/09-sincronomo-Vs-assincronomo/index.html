Síncrono vs Assíncrono

Sincrono

Quando falamos que um código é sincrono, estamos dizendo que esse codigo segue um fluxo linear de execução (DE CIMA PARA BAIXO)

Asincrono

Já o assincrono é diferente, é como se ele se descolasse do fluxo, e não vai necessariamente esperar a execução do codigo da linha anterior para que ela execute
