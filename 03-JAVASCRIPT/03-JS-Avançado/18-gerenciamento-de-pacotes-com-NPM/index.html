Gerenciamento de pacotes utilizando NPM

Para que serve um gerenciador de pacotes?

  • Gerenciar pacores e dependencias de pacotes do nosso projeto.
  • Um pacote nada mais é que uma biblioteca com codigos prontos pra gente usar.
  • Dependencia de pacotes. (as vezes vamos precisar baixar mais de um pacote, pq um depende do outro)
  • Pacotes tem vesões.
