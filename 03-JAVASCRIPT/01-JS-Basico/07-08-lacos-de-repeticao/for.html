FOR

Serve para execultar varias vezes o mesmo codigo / comando

Anatomia do FOR

for(let i = 0; i <5; i++){

console.log(i)

}

for(inicializador; condicao;incremento)

declaracao

  • Inicializador - É utilizado para inicializar a variavel
  • Condição - É a expressão que vai ser avaliada a cada uma das iterações do loop (se o resultado for TRUE) a declaração vai ser execultada
  • Encremento - Vai incrementar a variavel com mais um até chegar em uma condição pré definida
  • Ceclaração - Vai ser o bloco de cidigo que vai rodar todas as vezes que o FOR execultar
