Variaveis

Variaveis ocupam espaço no PC

Quanto mais variaveis mais espaço ele vai ocupar

Para que serve as Variaveis?

Servem para armazenar valores e poder usar as variaveis em todo nosso codigo

Como declarar uma variavel?

temos 3 modos de declarar uma variavel

  • VAR
  • CONST
  • LET
Letras maiusculas e minusculas tem diferença, tmb não podemos criar nome com espaço Mas podemos utilizar o CamelCase, por exemplo nomeDoCliente

tmb não é aconselhado colocar acento

VAR

Primeiro definimos o tipo da Variavle

Var

depois damos um nome para a variavle (é aconselhado que seja um nome bem explicativo)

var nomePessoa = "Samio"

o = aqui é utilizado para atrubuir valores, para comparação utilizamos o == ou === Clique aqui para saber mais

quando colocamos aspas duplas estamos informando que a variavel é de tipo texto

CONST

A diferença do const pro var é que quando criamos uma variavel tipo CONST depois não podemos mudar o valor daquela variavel, ela se torna uma variavel CONSTante

cosnt emailPessoa = "s.a.m.i.o@hotmail.com"

emailPessoa = "samiorock@gmail.com"
