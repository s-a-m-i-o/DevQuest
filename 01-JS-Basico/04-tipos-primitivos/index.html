Tipos Primitivos

Tipos primitivos são variaveis dentro do Js

É a estrutura fundamental do codigo

O Js tem seis tipos de tipos primitivos

  • STRING
  • NUMBER
  • BOOLEAN
  • NULL
  • UNDEFINED
  • SYMBOL (novo) - não é tão usado

String

String são usados para armazenar e manipular texto, e pode ser escrito tanto com aspas simples ou duplas

Exemplo

let nome = "Samio"

Number

Para colocar número ele tem que estar sem aspas

Exemplo

let numero = 34

Bolean

Representa valor de verdadeiro (true) ou falso (false)

Exemplo

let usuarioVerificado = true ou false

Null

Representa a ausencia intencional de qualquer valor ou objeto

Exemplo

let pessoa = null

Undefined

Representa valor não atribuido

Basicamente - quando o Js não consegue definir o tipo da variavel

Exemplo

Geralmente é o proprio Js vai dizer que o valor é Undefined
