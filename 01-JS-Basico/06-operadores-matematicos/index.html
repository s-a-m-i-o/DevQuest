Operadores Matematicos

Servem para fazer operações matematicas

aperte a tecla F12

+

Soma

Serve para somar valores

let primeiroValor = 4

let segundoValor = 2

console.log(primeiroValor + segundoValor)

6

-

Subtração

Serve para subtrair valores

let primeiroValor = 4

let segundoValor = 2

console.log(primeiroValor - segundoValor)

2

/

Divisão

Serve para dividir valores

let primeiroValor = 4

let segundoValor = 2

console.log(primeiroValor / segundoValor)

2

*

Multiplicação

Serve para multiplicar valores

let primeiroValor = 4

let segundoValor = 2

console.log(primeiroValor * segundoValor)

8

%

Resto

Serve para mostrar o resto dos valores

let primeiroValor = 17

let segundoValor = 2

console.log(primeiroValor % segundoValor)

1
