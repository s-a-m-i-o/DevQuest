Console e Comentario no Js

Console.log

Serve para imprimir mensagens no console do navegador

console.log("Olá Mundo")

Comentario no JS

Existe duas formas de comentarios no JS

O comentario em bloco

Comentario de bloco utilizamos /* aqui vai o comentario */

Comentario em linha

Comentario em linha utilizamos o // essa linha sera comentada

Lembrando que o CTRL+; é o atalho para comentario
